# Human Lab — Science-Backed Experiment Canvas v1

**Purpose:** A cooked markdown library of Human Lab categories, experiment actions, fun experiment names, and scholarly science sources.

**Important filter:** This list favors experiments with reasonable support from peer-reviewed research. Some interventions are backed by direct trials; others are backed by strong adjacent evidence, such as behavior change, sleep hygiene, nutrition, exercise, or social psychology studies. Avoid marketing claims like “miracle supplement,” “detox,” or “biohack cure.”

# Notes for Cursor / Product Implementation

1. **Do not display dense citations in the main card.** Show a small “Science-backed” badge and put sources inside the detail screen.
2. **Use claim strength labels:** `Direct trial`, `Systematic review`, `Adjacent evidence`, `Curiosity only`.
3. **Supplement experiments need extra caution.** Magnesium has some human sleep evidence. Mushroom coffee, grounding, nutritional yeast, turmeric, and no-salt claims need tighter sourcing before being marketed as proven.
4. **Best initial categories for launch:** Sleep, Stress, Mood, Confidence, Relationships, Fitness, Nutrition, Money, Learning, Planet Saver, Entrepreneurship, Curiosity Lab.
5. **Best vibe:** not a chore list; each item should feel like a tiny mission.
